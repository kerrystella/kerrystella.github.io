原文出处：https://www.programcreek.com/2013/04/why-string-is-immutable-in-java/

在Java中String类是不可变的(final)。简单来说，一个不可变的类就意味着他的实例是不可修改的，实例的所有信息都是在实例创建的时候被初始化并且不可被修改。不可变类的设计有很多优点。这篇博文主要从内存，同步和数据结构的角度来具体说明这种不可变的概念。

决策准绳之我的价值观

发表于 2020-01-01 更新于 2022-08-25 分类于 人生感悟

价值 - MUST

价值涵盖三个层面：客户价值、组织价值和个人价值。

  • 客户价值

    宏观层面是降本增效，微观层面可以体现为缩短流程、减少人工参与、避免用户误操作。具体分析客户价值的时候需要结合微观和宏观综合考虑

  • 组织价值

    具体体现为带给企业和团队的增益。可以是增加销售额，降低实施成本，也可以是提升团队技能或者增强团队凝聚力

  • 个人价值

    从自身出发考虑行动的收益。可以是收入的增加，可以是技能的提升，也可以是提升幸福感
